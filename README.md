## ✨ Fitur

- **💻 Desain Responsif:** Tampilan yang menyesuaikan di berbagai perangkat, dari ponsel hingga layar lebar.

## 🛠️ Teknologi yang Digunakan

- **HTML5**: Struktur dasar dan konten halaman.
- **CSS3**: Styling dan layout yang responsif.
- **JavaScript**: Menambah interaktivitas dan dinamika pada halaman.

## ⚙️ Instalasi

Ikuti langkah-langkah berikut untuk menjalankan proyek ini secara lokal:

1. **Clone repository**:
    ```bash
    git clone https://github.com/usernamekamu/jujutsu-kaisen-webpage.git
    ```
2. **Masuk ke direktori proyek**:
    ```bash
    cd jujutsu-kaisen-webpage
    ```
3. **Jalankan halaman** dengan membuka `index.html` di browser favorit Anda. 🌐
